### 📜 **E-Factura**

**Sistema de Facturación Digital**

![Project Badge](https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)
![Project Badge](https://img.shields.io/badge/PostgreSQL-336791?style=for-the-badge&logo=postgresql&logoColor=white)
![Project Badge](https://img.shields.io/badge/Livewire-20232A?style=for-the-badge&logo=livewire&logoColor=61DAFB)

</div>

**🎯 Descripción:**  
E-Factura es un proyecto para implementar la **facturación electrónica en Colombia** según las normas de la **DIAN**, integrando las API de **Factus Halltec**. Automatiza y simplifica el proceso, asegura el cumplimiento legal y optimiza el flujo de trabajo de las empresas.

**⚡ Características:**

- 💼 Automatización del ciclo de facturación  
- 📜 Cumplimiento con normativa DIAN  
- 🔄 Integración con API de Factus Halltec  
- 📈 Optimización del flujo de trabajo
